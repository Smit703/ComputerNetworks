To compile :
make clients
make servers

To run :
Q1) ./UDPtimed 20000
    ./UDPtime localhost 20000

    Give a string line as input

Q2) ./UDPtimed1 21000
    ./UDPtime1 localhost 21000

    Set of instructions : add x y, mul x y, mod x y, hyp x y

Q3) ./TCPechod 12000
    ./TCPecho localhost 12000

    give input in format : filename.txt x(number of bytes)
    The output will be visible in filename1.txt file
